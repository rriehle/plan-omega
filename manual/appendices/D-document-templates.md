## Introduction

Consistent documentation is critical for long-term survival operations. These templates provide standardized formats for recording daily activities, tracking inventory, planning passages, assessing threats, and maintaining systems. Regular use of these forms creates an invaluable record of patterns, problems, and solutions.

**Usage Guidelines:**

- Print multiple copies and keep aboard in a dry, accessible location
- Laminate frequently-used templates for use with dry-erase markers
- Create digital backups of completed forms
- Review records regularly to identify patterns and trends
- Use forms as training tools for children to build responsibility

## Usage Guidelines

### Printing and Storage

1. **Print multiple copies** - Keep blanks aboard for future use
2. **Laminate frequent-use forms** - Use with dry-erase markers
3. **Create binders** - Organize by category with dividers
4. **Weatherproof storage** - Use waterproof bags/containers
5. **Digital backups** - Scan completed forms periodically

### Best Practices

1. **Consistency** - Use the same forms every time
2. **Completeness** - Fill out all sections, use "N/A" when not applicable
3. **Legibility** - Print clearly or type if possible
4. **Signatures** - Sign and date all forms
5. **Review** - Regularly review past records for patterns
6. **Update** - Revise templates as needs change

### Integration with Manual

These templates support the protocols described throughout the manual:

- **Daily Log** supports continuous monitoring (Chapter 22)
- **Inventory Tracking** supports P.A.C.E. planning (Chapter 3)
- **Maintenance Schedule** supports vessel systems (Chapter 7)
- **Drill Reports** support training protocols (Chapter 21)
- **Threat Assessment** supports monitoring systems (Chapter 2)
- **Destination Research** supports navigation planning (Chapter 4)
- **Passage Planning** supports seamanship (Chapter 12)
- **Medical Records** support healthcare systems (Chapter 9)
- **Watch Schedules** support operational protocols (Chapter 20)

### Customization

Feel free to modify these templates for your specific:

- Vessel type and size
- Crew composition
- Operating region
- Specific needs and preferences

The key is **consistent use** - templates are only valuable if used regularly and completely.
